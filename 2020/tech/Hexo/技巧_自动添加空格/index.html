插件方法

VS Code 插件库中找到一个叫做 “pangu” 的插件，将其安装。该插件有两种生效方法：

  1. 按下 VSC 的快捷指令，输入 “>pangu”，然后可以选择处理当篇文章。
  2. 另一个策略是在设置中打开保存自动处理，将 auto_space_on_save 改为 true，并在其默认处理的文件格式中增加 .md，然后重启 VSC，即可生效。
